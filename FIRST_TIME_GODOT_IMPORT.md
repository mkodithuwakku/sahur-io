# First-Time Godot Import

If you cloned this repository in VS Code, do not import the repository root into Godot.

Use the nested `sahur-io/` folder as the Godot project directory.

## Correct Project Directory

If your local clone is:

```text
/path/to/Sahur-io
```

Then the folder you should import into Godot is:

```text
/path/to/Sahur-io/sahur-io
```

That folder contains the actual Godot project file:

```text
/path/to/Sahur-io/sahur-io/project.godot
```

## In Godot

1. Open Godot.
2. Click `Import`.
3. Navigate to your cloned repository.
4. Open the nested `sahur-io/` folder.
5. Select `project.godot`.

## Important

- Do not select the top-level repository folder.
- Do not look for `project.godot` in the repository root.
- The correct Godot project lives inside the nested `sahur-io/` directory.
